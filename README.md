# Notesy - Study Companion

Notesy is a lightweight, minimal notes extension for Chrome, specifically designed for online study sessions.

## Features
- Take notes inside Chrome extension popup.
- Auto-save notes locally (2-second debounce).
- Manual save, clear notes, and insert timestamp buttons.
- Export notes to **PDF** and **DOCX** with custom filenames.
- Works completely offline.
- Modern, "Serene Indigo" UI with **Dark Mode** support.
- Keyboard shortcut (`Ctrl+Shift+N`) for quick access.

## Installation
1. Ensure the following library files are in the `libs/` folder:
   - `jspdf.umd.min.js`
   - `docx.umd.js`
   - `FileSaver.min.js`
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the `notesy-extension` folder.
